# Nominal Network Psychometrics for Multiple-Choice Distractor Analysis

This repository contains all code and simulation data for the paper:

> Olson, J. L., & Templin, J. (in press). Modeling Multiple-Choice Distractor
> Relationships Using Network Psychometrics. *Multivariate Behavioral Research.*

---

## Overview

This paper introduces a nominal network psychometrics approach for modeling
option-level response patterns on multiple-choice assessments. Each response
option is treated as a node, and conditional associations among options are
estimated using LASSO-regularized multinomial logistic regression to construct
a network that visualizes frequently co-selected distractors.

The repository contains:
- R scripts for two simulation studies (3 and 9 misconceptions)
- 6,000 simulated datasets (1,000 per condition × 6 conditions)
- R script for the real-data analysis (Algebra Concept Inventory)
- R script for all figures in the paper

## Reproducing the Analysis

### Prerequisites

Install the required R packages:

```r
install.packages(c("psych", "MASS", "glmnet", "dplyr", "qgraph",
                   "IsingFit", "ggplot2", "patchwork", "readxl"))
```

R version 4.3 was used for all analyses.

---

### Step-by-step workflow

Each simulation condition has its own set of three scripts (generate, estimate,
aggregate) following a consistent naming convention: `_C1`, `_C2`, `_C3`
correspond to Conditions 1, 2, and 3 respectively. Run the full pipeline
separately for each condition.

The example below uses Simulation 1, Condition 2. Replace `C2` with `C1` or
`C3` for the other conditions, and replace `sim1`/`3` with `sim2`/`9` for
Simulation 2.

---

#### Step 1 — Generate simulated data

Each generation script was submitted as a 1,000-job array on the University of
Iowa's Argon HPC cluster. Each job generates one dataset for one replication.

```bash
# HPC array submission (one job per replication)
Rscript simulation/sim1/generate/SICM3_generate_Argon_C2.R <replication_number>
```

To run a single replication locally:
```r
Rscript simulation/sim1/generate/SICM3_generate_Argon_C2.R 1
```

Output is saved to `data/simulation/sim1/condition2/ResponseMatrix3_C2_{c}.RData`.

> **Note on reproducibility:** No random seed was set during original data
> generation. The 1,000 datasets used in the paper are provided in
> `data/simulation/` and represent the exact data underlying all reported
> results. Re-running the generation scripts will produce different (but
> statistically equivalent) datasets.

**Condition differences — Simulation 1:**
- `_C1`: Profile probabilities taken directly from Bradshaw & Templin (2014),
  Table 1. The r ≈ 0.5 correlation between misconceptions 1 and 2 emerges from
  these empirical proportions rather than being set explicitly.
- `_C2`: Tetrachoric correlation matrix with r = 0.8 between misconceptions 1
  and 2; all other pairs fixed at 0.
- `_C3`: Same approach as C2 with r = 0.2.

**Condition differences — Simulation 2:**
- All three conditions use the tetrachoric approach. Within-set correlations
  are set to 0.5 (C1), 0.8 (C2), or 0.2 (C3) for misconception sets
  {1,4,7}, {2,5,8}, and {3,6,9}. Between-set correlations are fixed at 0.

---

#### Step 2 — Estimate networks

Each estimation script reads one dataset, fits the nominal network model,
classifies item-option pairs into relationship groups, and saves results.

```bash
Rscript simulation/sim1/estimate/network3_estimate_Argon_C2.R <replication_number>
```

Output is saved to `NetworkResults/rep{c}.RData` and contains `qgraphMatrix`,
`qgraph_clean`, and `qgraph_avg` for that replication.

---

#### Step 3 — Aggregate results

Run locally after all 1,000 HPC jobs for a condition are complete. Reads all
`NetworkResults/rep{c}.RData` files and produces the aggregated summary objects
used in Tables 2–3 and Figure 2.

```r
Rscript simulation/sim1/aggregate/aggregate_results_sim1_C2.R
```

Output is saved to `network_avgs3_2.RData` (Sim 1) or `network_avgs9_2.RData`
(Sim 2). These files are loaded by `figures_and_realdata.R` to produce Figure 2.

---

#### Real data analysis and figures

```r
Rscript real_data/figures_and_realdata.R
```

This script requires all six `network_avgs` RData files to be present (produced
by the six aggregate scripts above). It produces:

- `results/figures/panel_boxplot_figure.pdf` — Figure 2 (simulation boxplots)
- `results/figures/AlgInvAllOptions.pdf` — Figure 5A (ACI network, positive connections)
- `results/figures/AlgInvNoCorrect.pdf` — Figure 5B (ACI network, incorrect options only)

## Script Descriptions

### Simulation 1 (3 misconceptions)

| Script | Description |
|--------|-------------|
| `SICM3_generate_Argon_C1.R` | Generates one dataset using empirical misconception profile probabilities from Bradshaw & Templin (2014). HPC array job. |
| `SICM3_generate_Argon_C2.R` | Generates one dataset using tetrachoric correlation r = 0.8 between misconceptions 1 and 2. HPC array job. |
| `SICM3_generate_Argon_C3.R` | Same as C2 with r = 0.2. HPC array job. |
| `network3_estimate_Argon_C1/C2/C3.R` | Fits multinomial LASSO network on one dataset; classifies item-option pairs into relationship groups; saves per-replication group averages. HPC array job. |
| `aggregate_results_sim1_C1/C2/C3.R` | Reads all 1,000 NetworkResults files for one condition and computes overall group means and SDs (Table 2). Run locally. |

### Simulation 2 (9 misconceptions)

| Script | Description |
|--------|-------------|
| `SICM9_generate_Argon_C1.R` | Generates one dataset with 9 misconceptions in 3 correlated sets; within-set r = 0.5. HPC array job. |
| `SICM9_generate_Argon_C2.R` | Same as C1 with within-set r = 0.8. HPC array job. |
| `SICM9_generate_Argon_C3.R` | Same as C1 with within-set r = 0.2. HPC array job. |
| `network9_estimate_Argon_C1/C2/C3.R` | Same as Sim 1 estimation but uses item-group logic (items 1–10, 11–20, 21–30) to classify correlated vs. uncorrelated misconception pairs. HPC array job. |
| `aggregate_results_sim2_C1/C2/C3.R` | Reads all 1,000 NetworkResults files for one condition and computes overall group means and SDs (Table 3). Run locally. |

### Shared

| Script | Description |
|--------|-------------|
| `figures_and_realdata.R` | Loads all six `network_avgs` files and produces Figure 2 boxplots. Also runs the full nominal network analysis on the ACI real dataset and produces Figures 5A and 5B. |

## Data

### Simulation data

Each `.RData` file contains the following objects:

| Object | Sim 1 | Sim 2 | Description |
|--------|-------|-------|-------------|
| `itemResponse` | ✓ | ✓ | 1,000 × 30 data frame of simulated item responses (1–4) |
| `latents` | ✓ | ✓ | 1,000 × k matrix: column 1 = theta; columns 2–4 (Sim 1) or 2–10 (Sim 2) = misconception indicators |
| `itemParameters` | ✓ | ✓ | Nested list of SICM item parameters per option (intercept, ability effect, misconception effect) |
| `option_qmatrix` | ✓ | ✓ | 120 × 3 (Sim 1) or 120 × 9 (Sim 2) Q-matrix mapping item options to misconceptions |
| `corr3_1/2/3` | ✓ | — | 3 × 3 tetrachoric correlation matrix (Sim 1 C2 and C3 only; C1 uses empirical probs) |
| `corr9_1/2/3` | — | ✓ | 9 × 9 tetrachoric correlation matrix |
| `misconceptionProbs` | ✓ | ✓ | Vector of misconception pattern probabilities (length 8 for Sim 1, 512 for Sim 2) |
| `cutpoints` | ✓ | ✓ | Threshold values used for dichotomizing continuous latent traits |
| `corData` | C1 only | — | Observed tetrachoric correlations (Sim 1 Condition 1 only) |

### Real data (Algebra Concept Inventory)

The ACI data (`data/real/ACI_FormA_Spring2019.xlsx`) were collected by
Wladis et al. (2018) from the City University of New York and are included
here with permission. The dataset contains Form A responses from Spring 2019
(N = 165 after exclusion of invalid forms; N = 143 after item-level exclusions
required for 20-fold CV).

See `real_data/figures_and_realdata.R` for the full data cleaning and
exclusion procedure, including which items and options were removed and why.

## Simulation Design Summary

### Simulation 1: 3 Misconceptions

| Condition | Script suffix | Correlation method | Correlation (mis 1 & 2) | All other pairs |
|-----------|-------------|-------------------|------------------------|-----------------|
| 1 | `_C1` | Empirical profile probs (Bradshaw & Templin, 2014) | ~0.5 (emergent) | ~0 |
| 2 | `_C2` | Tetrachoric | 0.8 | 0 |
| 3 | `_C3` | Tetrachoric | 0.2 | 0 |

- 1,000 examinees, 30 items, 4 options per item (A = correct)
- Option B = misconception 1; C = misconception 2; D = misconception 3
- 1,000 replications per condition

### Simulation 2: 9 Misconceptions

| Condition | Script suffix | Within-set correlation | Between-set correlation |
|-----------|-------------|----------------------|------------------------|
| 1 | `_C1` | 0.5 | 0 |
| 2 | `_C2` | 0.8 | 0 |
| 3 | `_C3` | 0.2 | 0 |

- Misconception sets: Set 1 = {1, 4, 7}, Set 2 = {2, 5, 8}, Set 3 = {3, 6, 9}
- Items 1–10: B = mis 1, C = mis 2, D = mis 3
- Items 11–20: B = mis 4, C = mis 5, D = mis 6
- Items 21–30: B = mis 7, C = mis 8, D = mis 9
- 1,000 examinees, 30 items, 4 options per item (A = correct)
- 1,000 replications per condition
- Note: 2 datasets were omitted from analysis (1 from C1, 1 from C2) due to
  insufficient observations of a misconception pattern during 20-fold CV.

## HPC Notes

All scripts ending in `_Argon.R` were designed for the University of Iowa's
Argon HPC cluster and accept a single integer command-line argument (the
replication number, 1–1000). Each script was submitted as a 1,000-job array,
with one job per replication per condition.
